# few-shot-learning (少样本学习)   
少样本学习旨在通过少量的训练样本，运用神经网络，基于度量学习的思想，比较特征来进行学习。基本模型包括：(1)Siamese Neural Network(孪生网络)；(2)Matching Network(匹配网络)；(3)Prototypical Network(原型网络)；(4)Relation Network(关系网络)。
